다방면에서 1인분이 가능한 개발자가 되는 것이 목표입니다.
기본기를 다지고 있으며 아키텍쳐 측면에서 아이디어를 제시할 수 있고 구현 가능한 수준이 되려 노력중입니다.
NestJS는 프레임워크 및 메타데이터, 데코레이터를 능숙하게 다루는 수준이며 Prisma를 통해 데이터를 활용할 수 있습니다.
추가적인 CQRS, 메세지큐를 활용한 MSA 패턴을 공식문서를 보고 구현해본 경험이 있습니다.

Spring은 프레임워크의 추상화 구조를 이해하고 활용할 수 있습니다.
